## About

Quimi is a web application built with Swift and Hummingbird that provides insights and analysis on the economic relationship between Indonesia and Australia.

## Getting Started

### Prerequisites
- Swift 6.2 or later
- macOS 14 or later

### Running the Application

```bash
swift run
```

The server will start on `http://127.0.0.1:8080`
